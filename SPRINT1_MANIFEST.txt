ARCHITECTURE SUMMARY
================================================================================

COMPONENT HIERARCHY:
───────────────────
SimplifiedSlideEditor (main layout)
├─ Toolbar
├─ Navigator (optional)
├─ EditableSlideCanvas (main canvas)
│  ├─ ContentRenderer (renders markdown)
│  ├─ SelectableSlideElements (click overlays)
│  ├─ SelectionHandles (visual feedback)
│  └─ InlineElementEditor (floating panel, portal)
└─ MarkdownPanel (optional)

DATA FLOW:
──────────
User clicks → SelectableSlideElements detects → EditableSlideCanvas updates
selected element → SelectionHandles render → InlineElementEditor opens →
User edits → Markdown updates → PresentationStore updates → Slide re-renders

STATE MANAGEMENT:
─────────────────
Local: selectedElement (EditableSlideCanvas)
Global: slides[].markdownContent (PresentationStore)
Global: editingSlide, showNavigator, theme (UIStore)
