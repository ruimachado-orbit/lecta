# Welcome to Lecta

The open-source platform for **technical presentations** with live code execution.

## Features

- **Slides + Code** side by side
- **4 execution engines**: JavaScript, Python, SQL, and native
- **Artifacts**: Attach PDFs, Excel files, images to any slide
- **AI Speaker Notes**: Generate with Claude
- **File-based**: Author in your favorite editor, present in Lecta

## Getting Started

Navigate with **arrow keys** or click the arrows in the toolbar.

Press **Cmd+Enter** to run code. Press **F5** to enter presenter mode.
